Sidebar Annotations

  • Width: 224px → 48px (collapsed)

    Standard expanded width allows comfortable reading. Collapses to icon-only view via toggle button.

  • Drag Handles

    Appear on hover (6-dot icon). Allow drag-drop reordering of pages within sections and across folders.

  • Expand/Collapse

    Chevron icon rotates 90° when expanded. Click anywhere on folder row to toggle. Nested pages show with left indent.

  • Action Buttons

    Fade in on hover: + (add sub-page), ••• (more options: rename, delete, duplicate, move).

  • Active State

    3px left accent border + background highlight. Follows currently open page.

  • Keyboard Navigation

    ↑↓ navigate pages, → expands folder, ← collapses, Enter opens page, Backspace goes to parent.
